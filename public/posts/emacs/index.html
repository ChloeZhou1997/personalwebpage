Package System Setup

use-package

  • GitRepo for the documentations.
  • See quick tutorial for how to use use-package

use-package is a way to organize the code neat and also by defer loading of packages it will improve the start up time of emacs. Typical pacakges that implies defering loading of packages includes: :command , :bind, :hook, :defer, if one package is needed to be run at the start up time, using :demand to make sure the package is get loaded.
